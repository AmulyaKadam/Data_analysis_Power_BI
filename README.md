This project demonstrates the end-to-end data analysis and interactive
dashboard development process using Microsoft Power BI.\
The analysis is based on the Blinkit Grocery Sales Excel dataset and
provides actionable business insights through dynamic visualizations.

# 🧩 Detailed Overview

The Blinkit Grocery Sales Analysis project showcases the complete
Business Intelligence workflow:

1.  Data Collection (Excel dataset)
2.  Data Cleaning & Transformation using Power Query
3.  Data Modeling in Power BI
4.  KPI Creation using DAX
5.  Interactive Dashboard Design

The final dashboard enables stakeholders to evaluate sales performance,
outlet efficiency, product contribution, and customer satisfaction.

------------------------------------------------------------------------

# 🎯 Project Objective

To develop an interactive Power BI dashboard that:

-   Analyzes total sales performance
-   Compares outlet types and sizes
-   Evaluates product category contribution
-   Analyzes customer ratings
-   Identifies high-performing locations
-   Enables data-driven business decisions

------------------------------------------------------------------------

# 🏗️ Dashboard Architecture

The dashboard consists of:

• KPI Cards (Total Sales, Avg Sales, No. of Items, Avg Rating)\
• Outlet Establishment Trend (Year-wise Sales)\
• Fat Content Analysis\
• Item Type Sales Distribution\
• Outlet Size Comparison\
• Outlet Location Tier Analysis\
• Outlet Type Performance Table

------------------------------------------------------------------------

# 🔄 Data Flow

1.  Raw Data Imported from Excel
2.  Data Cleaning using Power Query
    -   Removed inconsistencies
    -   Formatted data types
    -   Handled missing values
3.  Data Modeling
    -   Established relationships
4.  DAX Measures Created
    -   Total Sales
    -   Average Sales
    -   Total Items
    -   Average Rating
5.  Interactive Dashboard Visualization

# 📊 What I Analyzed

## 1️⃣ Sales Performance

-   Total Revenue: \$1.20M
-   Peak Sales Year: 2018 (\$205K)
-   Stable Growth Range: \~\$130K (2012--2017)

## 2️⃣ Outlet Location Analysis

-   Tier 3: \$472.13K (Highest Contributor)
-   Tier 2: \$393.15K
-   Tier 1: \$336.40K

Tier 3 locations generate \~40% more revenue than Tier 1.

## 3️⃣ Outlet Size Contribution

-   Medium: \$507.90K
-   Small: \$444.79K
-   Large: \$248.99K

Medium-sized outlets deliver highest revenue efficiency.

## 4️⃣ Fat Content Analysis

-   Low Fat: \$776.32K (\~65%)
-   Regular: \$425.36K (\~35%)

Low-fat products dominate total sales.

## 5️⃣ Top Item Categories

-   Fruits & Vegetables (\~\$0.18M)
-   Snack Foods (\~\$0.18M)
-   Household (\~\$0.14M)
-   Frozen Foods (\~\$0.12M)
-   Dairy (\~\$0.10M)

## 6️⃣ Outlet Type Performance

  Outlet Type         Total Sales   No. of Items   Avg Sales   Avg Rating
  ------------------- ------------- -------------- ----------- ------------
  Grocery Store       \$151.94K     1083           \$140       4
  Supermarket Type1   \$787.55K     5577           \$141       4
  Supermarket Type2   \$131.48K     928            \$142       4
  Supermarket Type3   \$130.71K     935            \$140       4

Supermarket Type1 contributes \~65% of total revenue.

------------------------------------------------------------------------

# 📌 Key Business Insights

-   Revenue exceeded \$1.20M across 8,523 products.
-   Supermarket Type1 dominates sales contribution.
-   Tier 3 locations show strongest performance.
-   Medium outlets outperform large outlets.
-   Low-fat products generate nearly double revenue compared to regular
    products.
-   2018 was the highest-performing year.
